## 발견된 문제 (2건)

### Issue 1: trimesh Path2D API 호환성
- **증상**: `paths_to_polygons([path])` → `TypeError: object of type 'Path2D' has no len()`
- **원인**: trimesh 4.x에서 `paths_to_polygons` 함수 시그니처 변경
- **우회**: `trimesh.creation.box()`로 bounding box 사각형 생성. 비정형 평면도 형태 미반영.
- **이슈 파일**: `reports/issues/trimesh_floor_extrude_api.md` (미해결)

### Issue 2: scipy 미설치
- **증상**: `ModuleNotFoundError: No module named 'scipy'`
- **원인**: trimesh의 face_colors → vertex_colors 변환에 scipy.sparse 필요
- **해결**: `pip install scipy`, `requirements.txt`에 추가
- **이슈 파일**: `reports/issues/scipy_missing_for_trimesh_solved.md`
