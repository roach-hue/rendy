## 수정 사항 (테스트 중 발견)

1. **wall_facing 회전각 오류**: 법선 각도(-90°) 사용 → bbox가 벽 바깥으로 확장. 벽 방향 각도(0°)로 수정.
2. **floor polygon contains 판정**: 벽면 slot은 경계선 위라 strict contains 실패. 95% 면적 겹침 허용으로 변경.
3. **placed_polygons key 불일치**: `polygon` vs `bbox_polygon` → `bbox_polygon`으로 통일.
4. **inward/center 벽면 오프셋**: slot이 벽 위(x=0)일 때 bbox 50%가 floor 밖. 법선 방향 500mm 오프셋 추가.
